MATLAB Project Wizard Wizard will generate a project and will 
request you to pick up a top level file to add to the project.
You can later use the M++ button from the Visual MATLAB Addin 
to add more files to the project.

The project contains:
 
main.c		Main C MEX file function
Read_ADC.dsw		MSVC Developer Studio Workspace
Read_ADC.dsp		MSVC Developer Studio Project File
readme.txt	This text
